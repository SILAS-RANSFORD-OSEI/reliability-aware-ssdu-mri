## Objective

Test the basic uncertainty utility functions using **synthetic stochastic reconstructions** derived from a real fastMRI brain zero-filled reconstruction.

> **Relationship to prior experiment:** This experiment repeats and consolidates the utility verification from [Experiment 007](EXP_007_ARTIFICIAL_UNCERTAINTY_UTILITY_TEST.md), confirming that the same functions behave correctly as part of the integrated pipeline.

---

## Motivation

The proposed reliability-aware reconstruction framework will eventually generate multiple stochastic reconstructions and compute a voxel-wise uncertainty map. Before implementing a stochastic neural reconstruction model, the basic uncertainty functions must be verified in isolation.

This experiment tests the following components:

| Component | Description |
|-----------|-------------|
| Mean reconstruction | Average across stochastic samples |
| Voxel-wise variance | Sample variance across stochastic samples |
| Uncertainty map normalisation | Scaling $U$ for downstream comparison |
| Uncertainty–residual alignment | Pearson correlation between $U$ and $E_{\mathrm{img}}$ |

---

## Method

A real fastMRI brain zero-filled reconstruction was used as the base image. Synthetic stochastic reconstructions were generated by adding independent Gaussian noise:

$$\hat{x}_t = \hat{x}_{ZF} + \epsilon_t, \qquad t = 1, 2, \ldots, T$$

| Term | Description |
|------|-------------|
| $\hat{x}_t$ | Synthetic stochastic reconstruction at sample $t$ |
| $\hat{x}_{ZF}$ | Zero-filled reconstruction |
| $\epsilon_t$ | Independent random Gaussian noise |
| $T = 8$ | Number of stochastic samples |

**Mean reconstruction:**

$$\bar{x} = \frac{1}{T} \sum_{t=1}^{T} \hat{x}_t$$

**Voxel-wise uncertainty map (sample variance):**

$$U = \frac{1}{T-1} \sum_{t=1}^{T} \left( \hat{x}_t - \bar{x} \right)^2$$

The uncertainty map $U$ was compared with the backprojected held-out residual energy $E_{\mathrm{img}}$ using **Pearson correlation**.

---

## Results

| Quantity | Value |
|----------|------:|
| Number of stochastic samples | `8` |
| Stochastic reconstructions shape | `8 × 768 × 396` |
| Mean image shape | `768 × 396` |
| Uncertainty map shape | `768 × 396` |
| Uncertainty–residual alignment (Pearson $r$) | `-0.0016` ≈ 0 |

---

## Interpretation

The near-zero Pearson correlation is the **expected result**. Because $U$ was generated from independent Gaussian noise rather than from a physics-informed reconstruction model, it appeared as random speckle with no spatial relationship to $E_{\mathrm{img}}$.

This confirms that meaningful uncertainty–residual alignment is **not a trivial property** of the pipeline — it must be produced by a model that understands MRI physics and k-space consistency.

---

## Importance for the Proposed Method

This experiment functions as a **negative control**, establishing the floor that any learned uncertainty model must surpass.

| Reconstruction type | Expected $U$–$E_{\mathrm{img}}$ alignment |
|---------------------|------------------------------------------|
| Artificial Gaussian noise (this experiment) | ≈ 0 — no meaningful alignment |
| Proposed stochastic model (future) | > 0 — model-informed spatial alignment |

For the proposed method to be meaningful, the model-derived uncertainty map must align with $E_{\mathrm{img}}$ significantly better than this random baseline.

---

## Conclusion

The uncertainty utility functions — mean reconstruction, voxel-wise variance, and normalisation — are verified and work correctly as part of the integrated pipeline.

However, artificial random uncertainty carries no reliability information and does not align with the residual-based signal.

**Next step:** replace the artificial noise with a model-based stochastic reconstruction mechanism — such as Monte Carlo dropout or an ensemble-based model — so that $U$ reflects genuine reconstruction uncertainty rather than random perturbations.
