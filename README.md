# reliability-aware-ssdu-mri
Reliability-aware self-supervised k-space MRI reconstruction with calibrated voxel-wise uncertainty for accelerated brain MRI.



# Reliability-Aware Self-Supervised MRI Reconstruction

This project investigates self-supervised accelerated brain MRI reconstruction using held-out k-space consistency for both reconstruction learning and uncertainty calibration.

The goal is to reconstruct an MR image from undersampled k-space and estimate a voxel-wise uncertainty map that indicates regions where the reconstruction may be unreliable.

The framework is inspired by SSDU-style self-supervised reconstruction, physics-guided data consistency, and uncertainty-aware medical imaging AI.
